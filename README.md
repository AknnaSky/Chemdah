# 构建

* [Gradle](https://gradle.org/) - Dependency Management

The GradleWrapper in included in this project.

On Windows:

```
gradlew.bat clean build
```

On MacOS/Linux:

```
./gradlew clean build
```

Build artifacts should be found in `./build/libs` folder.
